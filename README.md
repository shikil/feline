# feline

## Example of Functional Programming in Lean

### Build instructions

1. Download <https://github.com/lean-dojo/LeanCopilot/releases/download/v4.28.0/arm64-macOS.tar.gz>
2.

```bash
mkdir -p .lake/packages/LeanCopilot/.lake/build/lib
tar zxf ~/Downloads/arm64-macOS.tar.gz .lake/packages/LeanCopilot/.lake/build/
cp .lake/packages/LeanCopilot/.lake/build/lib/libctranslate* ~/.elan/toolchains/leanprover--lean4---v4.28.0/lib/
```
